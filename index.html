Intro

Showcase for my personal CSS library. Naturally, this is nowhere near as elegant as some of the larger frameworks out there, but it gets the job done for me and my projects. This example is just a very generic setup showcasing some basic layout classes and widgets I put in the page. A real project would obviously add styles onto these as needed.

*I mainly just uploaded this to GitHub so I'd have an easy to access point for the main files that I could update and download as needed.
